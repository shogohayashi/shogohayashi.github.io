Error Accessing object of type TableName which has been invalidated or deleted

React Native+realmでrealmObjectでFlatListを生成してるとき、deleteをするとタイトルのエラーが出てくる。

どうやらFlatListにrefを貼っているとrealmへの参照が効きっぱなしで残っており、削除済みのobjectへアクセスしてしまってエラーになるらしい

改善

realmのデータベースの参照を外せば良い.
